# Playwright Automated Tests - Place a Bet on a Sports Event

Automation tests for **Place a Bet on a Sports Event** with Playwright (JavaScript).  

## Technology stack overview

- **Playwright** → последна стабилна версия (npm install @playwright/test)
- **Node.js** → v20 или по-нова
- **JavaScript** (ES modules)
- **Test runner** → вграден в Playwright (@playwright/test)
- **Browser engines** → Chromium (enabled by default), с възможност за Firefox и WebKit

## Setup instructions

1. Download the repo from:
   git clone https://github.com/lumpyto/Play_Jack
   
2. Install the dependancies:
    install node.js 
    napm install
    npx playwright install

3. Test execution guide:
    npx playwright test
